### Penulisan Algoritme 1
```javascript
Program Menampilkan Bilangan 1 Sampai 10
  Deskripsi : Program ini digunakan untuk menampilkan bilangan 1 sampai 10 menggunakan perulangan

Deklarasi
  Tetapkan variabel X dengan tipe data integer

Algoritme
 1. Beri nilai pada variabel X = 0
 2. Selama X lebih kecil sama dengan 10, maka
      Tambahkan X dengan nilai 1
      Cetak nilai X
 3. Selesai
```
