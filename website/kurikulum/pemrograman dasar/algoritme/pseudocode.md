### Penulisan Algoritme Dengan Komentar
```javascript
/*

 */
Program Bilangan Ganjil atau Genap
  Deskripsi : Program Bilangan Ganjil atau Genap digunakan untuk menentukan apakah suatu bilangan tergolong ganjil atau genap

/*
 * Pendeklarasian
 * @param Angka {Integer} variabel Angka dengan tipe data integer
 */
Deklarasi
  Angka : Integer

/*
 * Proses / Badan / Body
 */
Algoritme
  Read(Angka) // Pengguna memasukkan nilai ke dalam variabel Angka
  If Angka mod 2 equal to 0 // penyeleksian apakah bilangan bulat sisa pembagian Angka sama dengan 0 
    then Write("Angka ini genap.") // Menampilkan hasil berupa teks
  Else Write("Angka ini ganjil.") // Menampilkan hasil berupa teks
```
